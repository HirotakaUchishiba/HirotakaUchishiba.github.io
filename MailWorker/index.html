Native macOS utility

MailWorker

A focused menu bar app that helps users organize Gmail, turn email into Google Calendar events, and review duplicate events before making changes.

Gmailの受信箱整理、メールからの予定登録、重複予定の確認を行うmacOSメニューバーアプリです。

User control

Review before change

Gmail cleanup

Preview archive rules, move approved unwanted messages to Trash, send a standard unsubscribe request, or create a sender filter.

Calendar registration

Extract an event proposal from selected mail and create it only after approval, except for explicitly enabled trusted-sender automation.

Duplicate review

List possible duplicate events for a selected period, then update or delete only the pair the user approves.

Google user data

Used only for visible app features

Gmail messages Read messages matching user-defined queries and apply only requested or approved mailbox actions.
Gmail filters Create a filter only for a sender the user explicitly approves.
Calendar events List, create, update, or delete events for registration and duplicate cleanup workflows.

OAuth tokens are stored in macOS Keychain. App state remains on the user's Mac. MailWorker has no developer-operated data server. Full details are in the Privacy Policy.
